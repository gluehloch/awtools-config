Utility für das Einlesen und Verwalten von Property-Dateien.

Einstiegspunkt ist die Schnittstelle GlueConfig. Von dieser Schnittstelle leiten sich die Klassen PropertiesGlueConfig (ein Wrapper für die Klasse java.util.Properties), SystemGlueConfig (ein Wrapper für den Zugriff auf System.getProperties()), sowie CombinedGlueConfig (kombiniert verschiedene GlueConfigs und definiert eine Hierarchie).
